# RobotApp 2

[RobotApp](https://github.com/marinasundstrom/RobotApp) ported to TypeScript/JavaScript.

Built with TypeScript, NodeJS, Express, Socket.IO, HTML, Canvas.

## Screenshot

![Screenshet](/images/screenshot.png)

## Development

Latest NodeJS required.

To build:

```sh
npm build
```

To run the program:

```sh
npm start
```

To run tests:

```sh
npm test
```
